# Azure Boards Work Items

## Epic

AI Service Desk Automation Platform

## Features

- Chatbot User Experience
- RAG Knowledge Base
- Ticket Automation
- Admin Dashboard
- CI/CD and Azure Deployment
- Security and Compliance
- Observability and Monitoring

## Example User Stories

- As an employee, I want to ask the AI assistant for IT help so I can resolve common issues without contacting service desk.
- As a service desk manager, I want ticket trends so I can identify recurring IT issues.
- As a platform engineer, I want CI/CD pipelines so every change can be tested and deployed consistently.
- As a security architect, I want secrets stored in Key Vault so sensitive data is not exposed.
